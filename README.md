# 🚀 E-commerce CI/CD Pipeline using Jenkins

## 🔧 Tech Stack

* Jenkins (CI/CD)
* Maven (Build Tool)
* SonarQube (Code Quality)
* Nexus (Artifact Repository)
* GitHub (Source Code)

## 🔄 Pipeline Stages

1. Code Checkout
2. Code Quality Analysis (SonarQube)
3. Build & Test
4. Artifact Packaging
5. Upload to Nexus
6. Deployment via SSH

## ✨ Features

* Pipeline as Code (Jenkinsfile)
* Quality Gate enforcement
* Automated artifact management
* Secure deployment using SSH

## 📌 Flow

GitHub → Jenkins → SonarQube → Maven → Nexus → Server Deploy

## 🎯 Outcome

Fully automated CI/CD pipeline ensuring faster and reliable deployments.
